# CO2Sensor4Arduino

This sketch was made to CO2 Sensor Device by Arudino micro.

## Preparation

1. [Download Adafruit_SSD1306](https://github.com/adafruit/Adafruit_SSD1306/archive/master.zip) library from GitHub.
2. Unzip and install.
